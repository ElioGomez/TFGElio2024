Proyecto Escuela - README
Descripción General
Este proyecto se divide en dos partes principales: el back-end y el front-end.

Back-End (escuela)
La carpeta escuela contiene toda la parte del back-end del proyecto. Esta parte está desarrollada utilizando Eclipse Jetty 9. 
Actualmente, las llamadas a la API han sido modificadas recientemente, por lo que podrían no estar funcionando correctamente. 

Front-End (web_instituto)
La carpeta web_instituto contiene todo lo relacionado con el front-end del proyecto. Esta parte esta desarrollada con React.
Aún faltan por aplicar algunos cambios de diseño y añadir algunas páginas y componentes adicionales. 

Prototipado
